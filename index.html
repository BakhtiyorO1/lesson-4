Урок 3

Продолжем изучение HTML.

Сегодняшный урок будет про элементов для форматирования текста.

1. Элемент < b > Задаёт полужирное начертания шрифта. Выделяет текст без акцента на его важность.

2. Элемент < em > Отображает шрифт курсивом, придавая тексту значимость.

3. Элемент < i > Отображает шрифт курсивом.

4. Элемент < small > Уменьшает размер шрифта на единицу по отношению к обычному тексту.

5. Элемент < strong > Задаёт полужирное начертание шрифта, относится к элементам логической разметки, указывая браузеру на важность текста.

6. Элемент < sub > Используется для создания нижних индексов. Сдвигает текст ниже уровня строки, уменьшая его размер.

7. Элемент < sup > Используется для создания степеней. Сдвигает текст выше уровня строки, уменьшая его размер.

8. Элемент < ins > Выделяет текст в новой версии документа, подчёркивая его. Для элемента доступны атрибуты cite и datetime.

9. Элемент < del > Перечёркивает текст. Используется для выделения текста, удаленного из документа. Для элемента доступны атрибуты cite и datetime.

10. Элемент < mark > Применяется для выделения фрагментов текста в справочных целях, окрашивая блок символов желтым цветом.
